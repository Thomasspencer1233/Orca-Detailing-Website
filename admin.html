☁ Cloud Sync Setup

Sync your jobs across all devices (phone + computer) using a free Firebase Realtime Database. Your data stays private in your own account.

  1. Go to console.firebase.google.com
  2. Create a new project (any name)
  3. Click Realtime Database → Create database
  4. Choose Start in test mode
  5. Copy the database URL (looks like: https://your-project-default-rtdb.firebaseio.com)
  6. Paste it below and tap Save
